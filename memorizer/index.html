  1. Enter the text you are trying to memorize or choose a preset if it is not already entered.
  2. Then enter the number of blanks. It defaults to the maximum number of blanks. Ususally you want to start at a low number.
  3. Click create.
  4. Enter the correct characters (case sensitive) into the boxes, using the mouse or tab to advance.
  5. When you think you're done, select "Show Answers" if you want it to and click "Check Answers".
  6. If you messed up, fix mistakes and check again.
  7. Repeat while increasing the number of blanks until the entire text is blanked.
  8. You should now be able to remember it.
  9. Celebrate!
